# Maxine’s World Balanced Subject Assets

This pack adds equal-weight visual concepts for Filipino, Philippine History/Makabansa, and GMRC.

## Contents

* Three isolated composition references. These are RGB images with a baked checkerboard, not transparent production sprites. Do not ship them directly.
* Three complete 16:9 scene plates suitable as destination references or temporary full-bleed scene art.
* `asset-metadata.json` with dimensions, role, alpha status, and SHA-256 hashes.

## Production gate

Normalize perspective, lighting, crop-safe regions, and apparent scale against all six subjects. Remove any pseudo-text. Obtain Filipino educator and cultural review before release. Keep UI labels and guide characters separate from scene art. Convert approved runtime art to WebP and retain layered source masters.
